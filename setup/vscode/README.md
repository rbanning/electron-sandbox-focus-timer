# Update `.vscode` Config

1) Add Custom **@** Directives for CSS
  - Copy `css-data.json` to the `.vscode/` folder
  - Update `.vscode/settings.json` to include the content of `settings-merge.json`

2) Add My Angular Snippets
  - Copy `my-angular-snippets.code-snippets` to `.vscode/` folder
  - Edit the snippets as needed (e.g. you may wish to use a different component selector prefix)
